# Go Algorithms & Practice

Этот репозиторий содержит мои решения базовых алгоритмических задач и сниппеты кода, написанные в процессе  изучения языка Go. 

## Описание

Здесь собраны решения задач, которые помогают мне:
* Нарабатывать мышечную память в синтаксисе Go;
* Изучать встроенные структуры данных (слайсы, мапы);
* Тренировать алгоритмическое мышление перед собеседованиями.

## Стек
* **Язык:** Go
* **ОС:** Ubuntu Linux
